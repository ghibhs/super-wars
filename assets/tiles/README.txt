# Adicione seus tiles em pixel art aqui!

Formatos recomendados:
- PNG (com transparência se necessário)
- Tamanho: 32x32 pixels (ou múltiplos de 32)

Depois de adicionar suas imagens:
1. No Godot, selecione o nó GameMap na cena world.tscn
2. Crie um novo TileSet
3. Adicione suas imagens de tile
4. Configure o tamanho do tile para corresponder às suas imagens
